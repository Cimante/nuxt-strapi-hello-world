# nuxt-strapi-hello-world

### Hello-world в среде Nuxt + Strapi + MongoDB

Порядок запуска:
1. Запустить .\mongo.exe
2. Запустить бэкенд на Strapi из папки /strapi:
```bash
$ npm run develop
```
3. Запустить фронт на Nuxt из корневой папки:
```bash
$ npm run dev
```
